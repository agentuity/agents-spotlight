# ✈️ Agno Travel Planner Agent

A sophisticated travel planning agent that creates detailed, multi-day itineraries tailored to user preferences, budget, and travel goals. This agent combines AI-powered travel intelligence with comprehensive destination knowledge to deliver personalized travel experiences.

## ✨ Key Features

- 🗺️ **Multi-Day Itineraries** - Detailed day-by-day travel plans
- 🏨 **Accommodation Suggestions** - Curated hotel and lodging recommendations
- 🎯 **Activity Planning** - Personalized attractions and experiences
- 🚗 **Transportation Logistics** - Travel routes and transportation options
- 💰 **Budget Optimization** - Cost-effective planning within specified budgets
- 🌍 **Local Insights** - Cultural tips and local recommendations
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **Agno Framework** - AI agent development framework
- **Exa Search** - Advanced travel and destination research
- **GPT-4o** - AI-powered itinerary generation and optimization
- **Agentuity SDK** - Cloud-native agent platform integration

### Travel Planning Workflow

1. **Goal Analysis** - Processes destination, group size, trip type, and budget
2. **Preference Mapping** - Identifies travel style and activity preferences
3. **Research Phase** - Uses Exa to gather destination information and options
4. **Itinerary Generation** - GPT-4o creates detailed multi-day plans
5. **Optimization** - Balances logistics, budget, and preferences
6. **Local Enhancement** - Adds cultural insights and practical tips

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Install required dependencies from requirements.txt:

```bash
pip install -r requirements.txt
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
