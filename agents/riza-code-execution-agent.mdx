# ⚡ Riza Code Execution Agent

A powerful AI agent that demonstrates secure code execution capabilities using Riza.io's sandboxed environment. This agent combines Claude's code generation with Riza's secure execution platform to run code safely in multiple programming languages.

## ✨ Key Features

- 🔒 **Secure Code Execution** - Safe code execution using Riza's sandboxed environment
- 🌐 **Multi-Language Support** - Execute Python, JavaScript, Ruby, PHP, and TypeScript
- 🤖 **LLM Integration** - Uses Claude with tool calling for intelligent code generation
- ⚡ **Real-time Execution** - Immediate code execution and result feedback
- 📊 **Output Handling** - Captures both stdout and stderr for comprehensive feedback
- 🛡️ **Sandboxed Environment** - Isolated execution prevents security risks
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **TypeScript** - Primary development language
- **Anthropic SDK** - Claude integration with tool calling
- **Riza.io API** - Secure code execution platform
- **Agentuity SDK** - Cloud-native agent platform integration

### Execution Workflow

1. **Prompt Processing** - Analyzes user requests for code generation needs
2. **Code Generation** - Uses Claude to generate appropriate code solutions
3. **Tool Calling** - Invokes Riza execution tool when code needs to be run
4. **Secure Execution** - Runs code in Riza's isolated sandbox environment
5. **Result Processing** - Returns execution results including output and errors

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Node.js**: Version 18 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)
- **Riza Account**: Sign up at [riza.io](https://riza.io)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Configuration

Set up your Riza API key:

```bash
agentuity env set --secret RIZA_API_KEY your_riza_api_key
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```

## 💻 Supported Languages

- **Python** - Full Python 3.x support with standard libraries
- **JavaScript** - Modern JavaScript with Node.js runtime
- **TypeScript** - TypeScript compilation and execution
- **Ruby** - Ruby scripting and application development
- **PHP** - PHP web development and scripting

## 🔒 Security Features

- **Sandboxed Execution** - Code runs in isolated containers
- **Resource Limits** - Controlled CPU and memory usage
- **Network Isolation** - Restricted network access for security
- **Time Limits** - Execution timeouts prevent infinite loops
- **Safe Environment** - No access to host system or sensitive data

## 🛠️ Tool Configuration

The agent configures Claude with a code execution tool that supports:

- **Language Detection** - Automatically identifies programming language
- **Code Validation** - Checks code syntax before execution
- **Error Handling** - Comprehensive error reporting and debugging
- **Output Capture** - Captures all execution output and errors

## 📊 Execution Results

Each code execution returns:

- **Standard Output** - Program output and print statements
- **Standard Error** - Error messages and warnings
- **Execution Status** - Success or failure indication
- **Runtime Information** - Execution time and resource usage
