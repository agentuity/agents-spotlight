# 📰 Startup News Scraper Agent

An intelligent automation system that combines AI-powered startup news analysis with workflow automation to deliver personalized daily digests across multiple communication channels. This project demonstrates advanced integration between Agentuity agents and external automation platforms.

## ✨ Key Features

- 🌅 **Automated Daily Digest** - Scheduled morning delivery at 7 AM
- 📡 **RSS Feed Processing** - Monitors multiple startup and tech news sources
- 🎯 **Smart Filtering** - AI-powered keyword filtering for startup-relevant content
- 📊 **Content Summarization** - Generates concise, structured summaries
- 📧 **Multi-Channel Distribution** - Email, Slack, and Discord delivery
- 🔄 **n8n Workflow Integration** - Seamless automation platform integration
- 🤖 **AI-Powered Analysis** - OpenAI integration for intelligent content processing
- 🚀 **Agentuity Native** - Built for cloud deployment and scaling

## 🛠️ Technical Implementation

### Core Technologies

- **TypeScript** - Primary development language
- **Agentuity SDK** - Cloud-native agent platform integration
- **OpenAI API** - AI-powered content analysis and summarization
- **n8n** - Workflow automation and scheduling
- **RSS Processing** - Real-time feed monitoring and parsing

### Automation Workflow

1. **Scheduled Trigger** - n8n triggers the workflow daily at 7 AM
2. **Agent Request** - HTTP POST to deployed Agentuity agent
3. **Feed Processing** - Agent fetches and filters RSS feeds for startup content
4. **AI Analysis** - OpenAI processes content for summaries and insights
5. **Content Formatting** - Structures data into HTML digest format
6. **Multi-Channel Delivery** - Distributes via Email, Slack, and Discord

## 🔧 n8n Workflow Setup

### 1. Import Workflow

1. Download `workflow.json` from the `n8n_workflow/` directory
2. Open n8n → **Workflows → Import from file**
3. Select the downloaded JSON file

### 2. Configure HTTP Request Node

- **Method**: POST
- **URL**: `https://agentuity.ai/api/<your-agent-id>?wait=true`
- **Headers**:
  - `Authorization`: `Bearer wht_XXXXXXXXXXXXXXXXXXXXXXXX`
  - `Content-Type`: `application/json`
- **Body**: `{}` (empty JSON object)

### 3. Set Up Communication Channels

#### Email Configuration
- **SMTP Credentials**: Configure your email provider
- **Subject**: `📬 Daily Startup Digest – {{ new Date().toLocaleDateString() }}`
- **Format**: HTML
- **Content**: `{{$json["html"]}}`

#### Slack Configuration
- **OAuth Bot Token**: Configure Slack workspace access
- **Channel**: Select target channel or user
- **Message**: `{{$json["html"]}}`

#### Discord Configuration
- **Bot Token/Webhook**: Configure Discord server access
- **Channel**: Select destination channel
- **Message**: `{{$json["html"]}}`
