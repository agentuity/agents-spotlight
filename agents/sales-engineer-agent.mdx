## ğŸ—ï¸ Multi-Agent Architecture

### 1. Sales Engineer Agent (Main)
The primary orchestrator that manages the entire RFP workflow:
- **Conversation Management** - Maintains dialogue state and context
- **Agent Coordination** - Delegates tasks to specialized agents
- **User Interaction** - Handles natural language communication
- **Workflow Control** - Manages the end-to-end RFP completion process

### 2. RFP Schema Agent
Specialized in document analysis and structure extraction:
- **Document Processing** - Analyzes uploaded RFP documents
- **Schema Extraction** - Identifies required fields and structure
- **Format Support** - Handles various document formats (PDF, etc.)
- **Structure Analysis** - Creates JSON schema from document requirements

### 3. RFP Generator Agent
Focused on document creation and completion:
- **Document Generation** - Creates completed RFP documents
- **Template Processing** - Uses extracted schema for document structure
- **Content Integration** - Incorporates conversation data into final document
- **Format Output** - Generates markdown-formatted RFP responses

### Core Technologies

### RFP Workflow

1. **Document Upload** - User provides RFP document (base64 encoded)
2. **Schema Extraction** - RFP Schema Agent analyzes document structure
3. **Conversational Gathering** - Sales Engineer Agent collects required information
4. **Information Processing** - System organizes and validates collected data
5. **Document Generation** - RFP Generator Agent creates completed document
6. **Final Delivery** - User receives completed RFP in markdown format

## ğŸš€ Quick Start with Agentuity

### Prerequisites

- **Node.js**: Version 18 or higher
- **Bun**: For package management and CLI execution
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

1. Install dependencies for the agents:
```bash
cd agent && bun install
```

2. Install dependencies for the CLI:
```bash
cd app && bun install
```

### Project Import

Import the project into your Agentuity account:

```bash
agentuity project import
```

### Development Mode

Start the agents in development mode:

```bash
cd agent && agentuity dev
```

Start the CLI interface:

```bash
cd app && bun run app
```
