# 🧠 CrewAI Social Media Agent

A sophisticated multi-agent system built with CrewAI that generates professional LinkedIn-ready social media content. This agent demonstrates advanced multi-agent coordination with specialized roles for research, writing, and editing.

## ✨ Key Features

- 🤖 **Multi-Agent Architecture** - Coordinated system with researcher, writer, and editor agents
- 📝 **LinkedIn Content Generation** - Creates professional, engaging social media posts
- 🔍 **Web Research Integration** - Uses Serper API for real-time content research
- 📊 **Structured Workflow** - Organized CrewAI task delegation and execution
- 🎯 **Topic-Focused Content** - Generates content based on user-defined topics
- 📚 **Source Attribution** - Includes references and links to original sources
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🏗️ Multi-Agent Architecture

### Researcher Agent
Specialized in gathering relevant information and insights:
- **Web Search** - Uses Serper API to find relevant articles and sources
- **Content Analysis** - Analyzes and summarizes key insights
- **Source Validation** - Ensures credibility and relevance of sources
- **Data Extraction** - Pulls key quotes and statistics

### Writer Agent
Focused on creating engaging social media content:
- **Content Creation** - Generates LinkedIn-style posts with proper formatting
- **Tone Optimization** - Maintains professional yet engaging voice
- **Structure Design** - Creates well-organized posts with clear messaging
- **Call-to-Action** - Includes appropriate engagement prompts

### Editor Agent
Ensures quality and polish of final content:
- **Content Review** - Reviews and refines generated posts
- **Quality Assurance** - Ensures professional standards and accuracy
- **Format Optimization** - Optimizes for LinkedIn's platform requirements
- **Final Polish** - Adds finishing touches and improvements

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **CrewAI** - Multi-agent orchestration framework
- **Serper API** - Web search and content discovery
- **OpenAI** - AI-powered content generation and analysis
- **Agentuity SDK** - Cloud-native agent platform integration

### Content Generation Workflow

1. **Topic Processing** - Analyzes user-provided topic for research direction
2. **Research Phase** - Researcher agent gathers relevant articles and insights
3. **Content Creation** - Writer agent generates initial LinkedIn posts
4. **Editorial Review** - Editor agent refines and polishes content
5. **Output Generation** - Delivers formatted posts with sources and references

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)
- **Serper Account**: Sign up at [serper.dev](https://serper.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Create and activate virtual environment:

```bash
source .venv/bin/activate
```

Install required dependencies:

```bash
pip install crewai crewai-tools python-dotenv uv
```

### Configuration

Set up your Serper API key:

```bash
agentuity env set --secret SERPER_API_KEY your_serper_api_key
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
