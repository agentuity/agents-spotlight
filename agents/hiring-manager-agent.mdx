# 🤝 Hiring Manager Agent

An innovative AI-powered interview system that demonstrates agent-to-agent communication by evaluating other AI agents through dynamic conversations. This sophisticated system showcases advanced multi-agent architectures and evaluation frameworks.

## ✨ Key Features

- 🤖 **Agent-to-Agent Communication** - Direct communication between AI agents
- 📋 **Dynamic Interview Process** - Adaptive questioning based on responses
- 🎯 **Comprehensive Evaluation** - Multi-criteria assessment framework
- 🔐 **Secure Authentication** - Registered applicant system with unique keys
- 💾 **State Persistence** - Maintains conversation history and context
- 📊 **Detailed Reporting** - Generates comprehensive evaluation reports
- 🔄 **Resume Capability** - Continues interviews from where they left off
- 🛡️ **Admin Controls** - Registration and management capabilities

## 🛠️ Technical Implementation

### Core Technologies

- **JavaScript** - Primary development language
- **Agentuity SDK** - Cloud-native agent platform integration
- **KV Storage** - Persistent state management and authentication
- **Multi-Agent Architecture** - Coordinated agent communication patterns

### Interview Workflow

1. **Applicant Registration** - Secure registration with unique authentication keys
2. **Interview Initiation** - Dynamic conversation startup with context setting
3. **Strategic Questioning** - Adaptive questions testing various capabilities
4. **Response Evaluation** - Real-time assessment of agent responses
5. **Comprehensive Scoring** - Multi-dimensional evaluation across 6 criteria
6. **Report Generation** - Detailed evaluation reports with recommendations

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Node.js**: Version 18 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash

```

## 📊 Evaluation Framework

### Assessment Criteria (1-5 Scale)

- **Technical Knowledge** - Understanding of domain-specific concepts
- **Problem Solving** - Ability to analyze and solve complex problems
- **Communication** - Clarity and effectiveness of responses
- **Authenticity** - Genuine responses vs. fabricated information
- **Memory** - Consistency and context retention across conversation
- **Adaptability** - Flexibility in handling unexpected questions

### Scoring System

- **5: Excellent** - Strong, context-aware, convincing responses
- **4: Good** - Mostly strong with minor issues
- **3: Adequate** - Functional but lacks depth
- **2: Poor** - Shows confusion or inconsistency
- **1: Fail** - Clear fabrication or evasion
