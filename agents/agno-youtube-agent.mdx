# 🎥 Agno YouTube Agent

A powerful video analysis agent that transforms YouTube content into structured insights. This agent fetches video transcripts, analyzes content structure, and generates comprehensive summaries with precise timestamps for key segments.

## ✨ Key Features

- 📹 **Video Transcript Extraction** - Fetches and processes YouTube video transcripts
- 🔍 **Content Structure Analysis** - Inspects video metadata and organization
- ⏰ **Intelligent Timestamping** - Generates precise timestamps for key segments
- 📊 **Content Grouping** - Organizes related content into logical sections
- 📝 **Comprehensive Summaries** - AI-powered video breakdowns and insights
- 🎯 **Highlight Generation** - Identifies and extracts key moments
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **Agno Framework** - AI agent development framework
- **YouTube API** - Video data and transcript access
- **GPT-4o** - Advanced content analysis and summarization
- **Agentuity SDK** - Cloud-native agent platform integration

### Analysis Workflow

1. **Video Processing** - Extracts video metadata and transcript data
2. **Content Analysis** - Analyzes video structure and key topics
3. **Segment Identification** - Identifies distinct content sections
4. **Timestamp Generation** - Creates precise timestamps for key moments
5. **Summary Creation** - Generates comprehensive video breakdowns
6. **Insight Extraction** - Produces actionable insights and highlights

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Install required dependencies from requirements.txt:

```bash
pip install -r requirements.txt
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
