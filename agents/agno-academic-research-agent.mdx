# 🎓 Agno Academic Research Agent

A specialized research agent designed for scholarly research and academic reporting. This agent uses Exa's academic search tools to find the latest peer-reviewed literature, synthesize insights across disciplines, and produce well-structured, citation-rich reports.

## ✨ Key Features

- 📚 **Academic Literature Search** - Access to peer-reviewed journals and scholarly publications
- 🔬 **Cross-Disciplinary Analysis** - Synthesis of insights across multiple academic fields
- 📝 **Citation-Rich Reports** - Properly formatted academic citations and references
- 🎯 **Rigorous Methodology** - Academic standards for research and analysis
- 📊 **Evidence-Based Analysis** - Data-driven conclusions and recommendations
- 🌐 **Latest Research** - Access to current academic publications and findings
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **Agno Framework** - AI agent development framework
- **Exa Academic Search** - Specialized academic literature discovery
- **GPT-4o** - AI-powered academic analysis and synthesis
- **Agentuity SDK** - Cloud-native agent platform integration

### Academic Research Workflow

1. **Research Question Formulation** - Defines clear academic research objectives
2. **Literature Search** - Uses Exa to find relevant peer-reviewed sources
3. **Source Evaluation** - Assesses credibility and relevance of academic sources
4. **Cross-Disciplinary Analysis** - Synthesizes findings across multiple fields
5. **Evidence Synthesis** - Combines research findings into coherent analysis
6. **Academic Report Generation** - Creates properly formatted scholarly reports

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Install required dependencies from requirements.txt:

```bash
pip install -r requirements.txt
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
