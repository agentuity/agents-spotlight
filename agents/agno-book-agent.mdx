# 📚 Agno Book Recommender Agent

A sophisticated literary recommendation agent that delivers personalized book suggestions based on favorite books, genres, and reading preferences. This agent combines semantic search with AI analysis to provide detailed book recommendations with comprehensive information.

## ✨ Key Features

- 📖 **Personalized Recommendations** - Tailored suggestions based on reading history
- 🔍 **Semantic Search** - Advanced book discovery using Exa search technology
- 🤖 **AI-Powered Analysis** - GPT-4o generates detailed literary insights
- 📊 **Comprehensive Details** - Titles, authors, ratings, summaries, and content tags
- 🎯 **Genre-Based Matching** - Finds books based on preferred genres and themes
- ⭐ **Quality Filtering** - Focuses on highly-rated and critically acclaimed books
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **Agno Framework** - AI agent development framework
- **Exa Search** - Semantic search for book discovery
- **GPT-4o** - AI-powered book analysis and recommendation generation
- **Agentuity SDK** - Cloud-native agent platform integration

### Recommendation Workflow

1. **Reading Profile Analysis** - Processes user's favorite books and genre preferences
2. **Preference Extraction** - Identifies desired themes and literary characteristics
3. **Semantic Search** - Uses Exa to find relevant books from literary databases
4. **AI Analysis** - GPT-4o analyzes and ranks potential recommendations
5. **Detail Enhancement** - Enriches recommendations with comprehensive book information
6. **Personalized Delivery** - Presents tailored recommendations with detailed insights

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Install required dependencies from requirements.txt:

```bash
pip install -r requirements.txt
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
