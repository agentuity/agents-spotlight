## ğŸš€ Quick Start with Agentuity

### Prerequisites

- **Bun**: Version 1.2.4 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

This command will open a browser window where you can log in to your Agentuity account.

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agents to the Agentuity Console in DevMode, allowing you to test and debug your multi-agent system in real-time.

### Configuration

Set up your environment variables for social media platform integrations:

```bash
agentuity env set TWITTER_API_KEY your_twitter_api_key
agentuity env set FACEBOOK_ACCESS_TOKEN your_facebook_token
agentuity env set LINKEDIN_ACCESS_TOKEN your_linkedin_token
agentuity env set INSTAGRAM_ACCESS_TOKEN your_instagram_token
```
