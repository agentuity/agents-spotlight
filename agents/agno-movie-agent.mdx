# 🎬 Agno Movie Recommender Agent

A sophisticated movie recommendation agent that delivers personalized film suggestions based on user preferences, favorite movies, and desired themes. This agent combines semantic search with AI analysis to provide detailed movie recommendations with comprehensive information.

## ✨ Key Features

- 🎭 **Personalized Recommendations** - Tailored suggestions based on user preferences
- 🔍 **Semantic Search** - Advanced movie discovery using Exa search technology
- 🤖 **AI-Powered Analysis** - GPT-4o generates detailed movie insights
- 📊 **Comprehensive Details** - Titles, ratings, genres, summaries, and more
- 🎯 **Theme-Based Matching** - Finds movies based on desired themes and moods
- 🌟 **Quality Filtering** - Focuses on highly-rated and critically acclaimed films
- 🚀 **Agentuity Native** - Built for seamless deployment on Agentuity platform

## 🛠️ Technical Implementation

### Core Technologies

- **Python** - Primary development language
- **Agno Framework** - AI agent development framework
- **Exa Search** - Semantic search for movie discovery
- **GPT-4o** - AI-powered movie analysis and recommendation generation
- **Agentuity SDK** - Cloud-native agent platform integration

### Recommendation Workflow

1. **Preference Analysis** - Processes user's favorite movies and genre preferences
2. **Theme Extraction** - Identifies desired themes and movie characteristics
3. **Semantic Search** - Uses Exa to find relevant movies from extensive databases
4. **AI Analysis** - GPT-4o analyzes and ranks potential recommendations
5. **Detail Enhancement** - Enriches recommendations with comprehensive movie information
6. **Personalized Delivery** - Presents tailored recommendations with detailed insights

## 🚀 Quick Start with Agentuity

### Prerequisites

- **Python**: Version 3.10 or higher
- **UV**: Version 0.5.25 or higher
- **Agentuity CLI**: Install from [agentuity.dev](https://agentuity.dev)

### Authentication

Before using Agentuity, you need to authenticate:

```bash
agentuity login
```

### Installation

Install required dependencies from requirements.txt:

```bash
pip install -r requirements.txt
```

### Development Mode

Run your project in development mode with:

```bash
agentuity dev
```

This will start your project and open a new browser window connecting your agent to the Agentuity Console in DevMode.

## 🌐 Deployment

When you're ready to deploy your agent to the Agentuity Cloud:

```bash
agentuity deploy
```
